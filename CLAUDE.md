# CLAUDE.md — проект North Macedonia Guidebook

Контентный сайт-справочник по всей Северной Македонии (столица — **Скопье**) **только на английском (en, на корне `/`)** — сайт одноязычный (ru/uk полностью удалены 2026-06-22): достопримечательности, города, еда (директория проверенных мест), маршруты, транспорт, аренда авто, страховки, развлечения, релокация, новости. Стратегия — **EN-only / Tier-1 / приоритет релокации-номадов / кластерная синергия / часть балканской сети** (Албания + Черногория + Хорватия + Македония). Места отбираются по высокому рейтингу (>4★ из нескольких источников) и сверяются с опытными путешественниками, гидами и местными жителями. Диверсифицированная партнёрская монетизация (экскурсии, жильё, аренда, страховки, eSIM) + медийная реклама на Tier-1-трафике. Единственный источник правды по продукту, нише и приоритетам — **SPEC.md**; стратегические решения — **docs/STRATEGY.md**, доказательная база — **docs/MACEDONIA-ANALYSIS-2026-06-21.md** (всё веб-проверено). Этот файл — правила работы внутри репозитория. При конфликте приоритет: SPEC.md → CLAUDE.md → STRATEGY.md → всё остальное.

## Стек и принципы
- **Astro 6 + Content Collections**, полная статика (SSG). Никаких SSR, БД, CMS.
- **Tailwind CSS 4**. Дизайн-токены — только из `src/styles/tokens.css` (SPEC §10). Хардкод hex-цветов в компонентах запрещён.
- **Без React.** Интерактив — нативный JS внутри Astro-компонентов, и только там, где без него нельзя: карта Leaflet, мобильное меню, переключатель языка.
- Хостинг: **Cloudflare Workers (Static Assets)**. Деплой = push в `main` (Workers Build: `npm run build` → `wrangler deploy`). Превью-деплои на ветках.
- Изображения: webp/avif через `astro:assets`, lazy-load, каждая ≤ 200 КБ.

## Железные правила
1. **Язык — только английский.** Сайт **одноязычный: en на корне `/`** (ru/uk удалены 2026-06-22). Весь новый контент создаётся ТОЛЬКО на en (`src/content/<коллекция>/en/<slug>.md`). Папок `/ru/`, `/uk/` нет; `LANGS = ['en']` в `src/i18n/types.ts` и `src/content.config.ts`. hreflang страницы — self `en` + `x-default` (оба на canonical). Прежние `/ru/*`, `/uk/*`, `/en/*` URL → 301 на корень (`public/_redirects`). Переключателя языка нет.
2. **Партнёрские ссылки** — только через `/go/{partner}?c={slug}` с `rel="sponsored nofollow noopener"`. Прямые URL партнёрок в контенте запрещены. На статью 1–3 AffiliateBox, не больше. Партнёрки и `partners.json` переиспользуются между сайтами балканской сети.
3. **URL не меняются** после индексации. Ошибся в slug — 301 в `public/_redirects`, а не переименование.
4. **Факты не выдумывать.** Цены, адреса, телефоны, условия — только из брифа `briefs/{slug}.md` или надёжного первоисточника. Нет данных → оставить `<!-- TODO: факт от владельца -->` и сказать об этом, а не сочинять. Рейтинги/отзывы из внешних источников (Google, TripAdvisor) дословно не перепечатывать — это нарушает их ToS; использовать как критерий отбора и ссылку, добавляя собственное описание (SPEC §15).
5. **YMYL-осторожность (визы/налоги/ВНЖ/бизнес).** Это чувствительные темы — факты ТОЛЬКО из официальных первоисточников (mfa.gov.mk, ujp.gov.mk, customs.gov.mk, PwC Tax Summaries и т.п.) + видимый дисклеймер «уточняйте, правила меняются». Известные ловушки, выявленные веб-аудитом: (а) **гражданам РФ нужна виза** с 21.03.2022 (исключение — держатели действующих виз US/UK/CA/Шенген C: безвиз 15 дней); US/UK/EU/Украина — безвиз 90/180; (б) **формальной визы цифрового кочевника в Македонии НЕТ** — «nomad visa Macedonia» в SEO-агрегаторах выдумка. Не публиковать как факт ничего из анализа с тегом «[требует сверки]».
6. **Перф-бюджет (мобайл):** LCP < 2.0 с, CLS < 0.1, JS на странице < 50 КБ. Любая новая зависимость тяжелее 10 КБ — спросить владельца.
7. **Доступность:** контраст ≥ 4.5:1, видимый focus, alt у каждого изображения, рабочая навигация с клавиатуры.
8. **SEO-обвязка каждой страницы:** title ≤ 60 зн., description ≤ 155, canonical, hreflang self `en` + `x-default` (сайт одноязычный), OG-картинка, schema.org по типу шаблона (SPEC §14).
9. **Сторонние скрипты:** только одна аналитика (SPEC §17). Никаких виджетов соцсетей, онлайн-чатов, тег-менеджеров. (Дисплей-реклама на Tier-1-трафике — открытое решение владельца, конфликтует с CSP/перф-духом; включать только по явному решению, STRATEGY §8 O5.)
10. **Доки не отстают от кода (правило владельца — обязательно).** После КАЖДОЙ доработки и ДО коммита синхронизировать документацию: `PROGRESS.md` (статус/роадмап + дата), `HANDOFF.md` (снимок: коммит, число страниц, «где остановились»); при изменении продукта/правил — `SPEC.md` / `STRATEGY.md` / этот `CLAUDE.md`; `AUDIT.md` — отметить закрытое; авто-память + её зеркало `docs/memory/`. **Доработка без актуализации доков НЕ считается завершённой.** Детали процесса — раздел «Синхронизация доков» ниже + память `update-docs-after-each-change`.

## Стратегия (что определяет приоритеты контента)
Подробности и обоснование — `docs/STRATEGY.md`, факты — `docs/MACEDONIA-ANALYSIS-2026-06-21.md`. Короткая рамка для повседневной работы:

- **EN-only / Tier-1.** Сайт одноязычный — только EN (корень `/`). Целевая аудитория — Запад через EN (US/CA/UK/Зап. Европа/AU): дорогой RPM, западные туристы и номады. RU/UK как каналы убраны (сайт переведён на один язык 2026-06-22).
- **Приоритет кластеров** (релокация/номады в приоритете — менее сезонно, дороже, Tier-1, рекуррентные партнёрки):
  1. **Планирование** (`/planning/`): виза, валюта-денар, когда ехать, как добраться — верх воронки, донор трафика.
  2. **MONEY-кластер:** `/car-rental/` · `/insurance/` · eSIM — транзакционные прямые `/go/`-связки.
  3. **Релокация/Номады** (`/relocation/`, EN-first): cost of living, регистрация компании (DOO/DOOEL), **плоские налоги 10%**, ВНЖ — приоритетный маржинальный рынок. Крючок — НЕ «номад-виза» (её нет), а низкий налог + дешёвая жизнь + кандидатство в ЕС + ВНЖ через работу/бизнес.
  4. **Достопримечательности / Города / Маршруты** — упор на **озеро Охрид** (ЮНЕСКО) и природу (нацпарки Маврово, Галичица, Пелистер). Лёгкий вход — **Битола** и вторичные точки (Крушево, Маврово, каньон Матка, Преспа).
  5. **Еда.** 6. **Развлечения.**
- **Кластерная (балканская) синергия.** Сайт — часть сети. Предусматривать кросс-линки на албанский/черногорский/хорватский сайты и общие мульти-страновые роуд-трип-страницы: **«Lake Ohrid — two-country guide»** (Охрид + Поградец, AL — кратчайший хоп ~40 км/1 ч), **«Balkan Grand Loop»** (MK→AL→ME→HR), **«Ohrid–Tirana–Adriatic»**.

## Ключевые факты страны (для контента; источник — docs/MACEDONIA-ANALYSIS-2026-06-21.md)
- **Страна/столица:** Республика Северная Македония, столица **Скопье**.
- **8 статистических регионов** (NUTS-3, верхний фильтр enum `region`): `Skopje` · `Polog` (центр Тетово) · `Eastern` (Штип) · `Northeastern` (Куманово) · `Southeastern` (Струмица) · `Southwestern` (**Охрид**) · `Vardar` (Велес) · `Pelagonia` (Битола).
- **Города/базы (enum + контент):** Skopje · Ohrid · Bitola · Tetovo · Struga · Mavrovo (курортная локация) · Kruševo (высокогорный, ~1350 м) · Prilep · Štip (+ опц. Debar, Strumica, Gostivar, Veles, Kumanovo).
- **Тур-якорь:** **озеро Охрид** — объект ЮНЕСКО смешанного типа (ref 99); одно из старейших (~1,2 млн лет) и глубочайших (288 м) озёр Европы. Муниципалитет Охрид = 28,8% туристов и 36,2% ночёвок страны; «озёрная» категория = 58,8% ночёвок. Резкая летняя сезонность (пик июль–август) → аффилиат-доход пикует Q3. Нацпарки: Маврово (пик Кораб 2 764 м — высшая точка страны), Галичица, Пелистер.
- **Валюта:** македонский денар (**MKD**), де-факто привязан к евро с 2002 — центральный паритет ~**61,5 MKD/EUR** (1 EUR = 61,3644 ±1%). Курс стабилен → переводы ден→EUR предсказуемы.
- **Визы:** US/UK/EU/Украина — безвиз 90/180; **РФ — нужна виза** (с 21.03.2022). Цифровой номад-визы НЕТ (см. правило 5). Регистрация в полиции в течение 48 ч (обычно делает отель).
- **Налоги:** **10% плоский НДФЛ**, **10% налог на прибыль (CIT)**, **18% НДС** (льготные 5%/10%), соцвзносы 28%. Бизнес-форма — DOO/DOOEL (аналог ООО), допускается единственный учредитель.
- **Масштаб рынка:** ~1,34 млн туристов / 2025 (рекорд, +6,0%), иностранцев ~68,5%, доминирует Турция. Рынок малый (≈в 4× меньше Грузии) → потолок трафика ниже, основной доход — медийка (Tier-1) + аффилиат, а не локальные продажи.

## Контент-конвейер (SPEC §12)
`briefs/{slug}.md` (ключевой запрос, вопросы читателя, ФАКТЫ владельца/первоисточника, цены, фото) → **скелет:** `npm run new -- <тип> <slug> --title "…"` (генерит en/ru/uk + папку фото + DoD-чеклист в комментарии) → черновик (EN-first — основной язык) по шаблону → фото (CC/свои) → переводы → **fact-check → content-editor** → тройка коммитится вместе, draft снимается. Порядок именно такой: сначала факты сверяются (`fact-checker`), потом перевод, потом финальный приёмщик (`content-editor`). Для YMYL-тем (релокация/визы/налоги) fact-check обязателен по первоисточникам.

## Стандарты контента (Этап 1, 2026-06-22)
- **Объём.** Статья — **1200–2000 слов** (первый абзац — прямой ответ за 2–3 предложения; H2/H3 под вопросы; без «воды», без FAQ-блока). Новость — **1500–2000 знаков** (~250–350 слов, 3–5 разделов, ≥4 внутр. ссылки).
- **Фото (минимум).** Статья ≥5 (cover + ≥4 инлайн); новость ≥2 (cover + ≥1); маршрут — cover + фото **каждой остановки**; город ≥10; столица Скопье ≥15. Каждое фото уникально на материал, **webp ≤200 КБ**, своё или CC/PD (Wikimedia/Openverse/Unsplash) с атрибуцией (автор + лицензия).
- **Город = статья `category=cities`** (английский слаг). Отдельная коллекция `cities` НЕ используется для наполнения. _(Имя `goroda` из движка-донора не применяется — слаги английские.)_
- **Тон.** Живой, человечный, как пишет носитель английского (сайт одноязычный, en). Не ИИ-генерёнка. Факты только из источника. Даты/время/числа — только из источника **текущего года** (для ежегодных событий никогда не копировать прошлогодние даты).
- **Партнёрки** — только через `/go/{partner}` с `rel="sponsored nofollow noopener"`. Прямые партнёрские URL в контенте запрещены.
- **Уникальность намерения (④ гейт сети).** Одна интент-страница: без mirror-дублей («A→B» = «B→A» = одна двунаправленная стр.) и тонкого variant-слайсинга (длины маршрутов / помесячные / «worth it» → разделы хаба); порог distinctness (уник. primary-keyword + непересекающиеся факты на 1200+ слов). Гейт `check-dedup` в `npm test` / `npm run qa`. Полные правила — хаб `docs/STANDARDS.md` → «④ Уникальность намерения»; память `one-intent-page-anti-cannibalization`.

## Путь публикации (унификация)
Любой контент проходит QA-гейты: `npm run qa` (check/build/test/links/lint + dist-аудит → GO/NO-GO), при изменении вёрстки/визуала — `npm run qa:browser` (Lighthouse mobile a11y/SEO/BP ≥90 + адаптив без горизонтальных переполнений). Фото — webp ≤200 КБ. Партнёрские ссылки — только `/go/{partner}`. Публикуем (снимаем `draft:false`) только при **GO**.
