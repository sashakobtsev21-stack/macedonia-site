## R4 — приоритеты контента (EN-first, см. `docs/STRATEGY.md`)
Порядок кластеров (релокация/номады — приоритет: менее сезонно, дороже, Tier-1, рекуррентные партнёрки):
1. **Планирование** (`/planning/`): виза, валюта-денар, когда ехать, как добраться — верх воронки, донор трафика.
2. **MONEY-кластер:** `/car-rental/` · `/insurance/` · eSIM — транзакционные прямые `/go/`-связки.
3. **Релокация/Номады** (`/relocation/`): cost of living, регистрация DOO/DOOEL, **плоские налоги 10%**, ВНЖ. Крючок — НЕ «номад-виза» (её нет, см. CLAUDE.md правило 5), а низкий налог + дешёвая жизнь + кандидатство в ЕС.
4. **Города/Маршруты/Достопримечательности** — якорь **озеро Охрид** (ЮНЕСКО), природа (Маврово, Галичица, Пелистер); лёгкий вход — Битола, Матка, Крушево.
5. **Еда.** 6. **Развлечения.**

**Балканская синергия:** кросс-линки на албанский/черногорский/хорватский сайты, мульти-страновые петли (Lake Ohrid two-country с Поградцем AL; Balkan Grand Loop MK→AL→ME→HR).

**Границы:** новые broad-travel темы — только с уникальной фактурой (свои фото/проверенные факты), без «SEO-генерёнки» (SPEC §15). YMYL (визы/налоги/ВНЖ) — только официальные первоисточники + дисклеймер. URL после индексации не менять (правка slug = 301 в `public/_redirects`).

### Сквозные правила (держать на всех фазах)
- **YMYL-осторожность** (визы/налоги/ВНЖ): факты только из первоисточников (`mfa.gov.mk`, `ujp.gov.mk`, `customs.gov.mk`, `stat.gov.mk`, PwC) + видимый дисклеймер. Зашитые ловушки: **РФ нужна виза** (с 21.03.2022); **номад-визы нет**; налог плоский **10%**; денар **~61,5/EUR**.
- **GSC → [ДАННЫЕ]:** без GSC/атрибуции приоритет — догадки. После запуска приоритизировать кластеры по реальному спросу.
- **Честные оговорки рынка:** ~1,34 млн туристов (в ~4× меньше Грузии) → потолок аффилиата ниже, локальные размещения — слабый канал; резкая сезонность (аффилиат пикует Q3) → приоритет несезонной релокации + evergreen-маршрутов. Основной доход — **медийка Tier-1 + аффилиат**.
- **Перф/CSP-дух:** перф-бюджет (LCP<2.0/CLS<0.1/JS<50КБ) — гейт. Дисплей и GA4 тянут consent и расширение CSP — только по решению O5.
