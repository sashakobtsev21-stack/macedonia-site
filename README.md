# North Macedonia Guidebook

Контентный сайт-справочник по **Северной Македонии** — **только на английском
(en, на корне `/`)**; сайт одноязычный (ru/uk удалены 2026-06-22): города,
достопримечательности, еда, маршруты, транспорт и т.д. Статика на Astro, хостинг
Cloudflare Workers.

> **Статус: каркас под Северную Македонию готов.** Репозиторий форкнут с движка
> **Georgia Guidebook** с обнулённой git-историей; грузинский контент снят, движок
> сохранён. **Перемоделировано под Македонию:** enum'ы (8 статрегионов / города /
> кухни в `content.config.ts`), i18n `en` (одноязычный), `SPEC.md` / `CLAUDE.md`, бренд
> (флаг `mk-flag.svg`, manifest, robots, GA отключён до своего ресурса), live-бар
> (погода Скопье/Охрид/Битола). **URL-слаги — английские** (EN-only): `/attractions/`,
> `/cities/`, `/food/`, `/entertainment/`, `/routes/`, `/relocation/` и т.д.
> Длинная i18n-проза (интро хабов, страховка, релокация, о-проекте) — пока
> **нейтральные заглушки** «раздел наполняется»; детальный текст — в контент-фазе.
>
> Коллекции контента почти пусты: 2 demo-сид-материала (lake-ohrid + north-macedonia-7-day-itinerary), остальное — `.gitkeep`; сайт
> собирается пустым по дизайну (SPEC §11). **Следующий шаг — контент** (города/
> достопримечательности/еда/маршруты/релокация по приоритетам `docs/STRATEGY.md`)
> (домен куплен; hero, иконки и og-картинка уже македонские).
> Грузинские референс-документы — в `docs/_georgia-reference/`.

**Домен:** `https://macedoniaguidebook.com` — ✓ **куплен** (ещё не задеплоен / не в GSC); конфиги
`astro.config.mjs` и `wrangler.jsonc` уже указывают на него.

## Стек

- **Astro 6** (полная статика SSG, без React) + **Content Collections**
- **Tailwind CSS 4**; дизайн-токены — только из `src/styles/tokens.css`
- Хостинг **Cloudflare Workers (Static Assets)** + Worker-роут `/go/` (`worker/index.ts`)
- Изображения webp/avif ≤ 200 КБ; PWA (установка + оффлайн)

## Команды

| Команда | Что делает |
|---|---|
| `npm run dev` | dev-сервер |
| `npm run build` | прод-сборка (проходит на пустых коллекциях) |
| `npm run check` | `astro check` — типы и схемы контента |
| `npm run lint` / `npm run format` | ESLint / Prettier |
| `npm test` | контент-гейты: enums · parity (en-only: lang↔папка, title ≤60) · photos · interlinks |
| `npm run qa` | финальный GO/NO-GO: чистая сборка + гейты + аудит `dist/` |

## Документация

- **`SPEC.md`** — продукт, шаблоны, контент-модель (источник правды; под Северную Македонию).
- **`CLAUDE.md`** — правила работы в репозитории.
- **`CONTENT_GUIDE.md`** — гайд по созданию контента.
- **`SETUP-GUIDE.md`** — подключение внешних агентов/плагинов (my_agents, ui-ux-pro-max).
- **`docs/STRATEGY.md`** — зафиксированные стратегические решения владельца.
- **`docs/ROADMAP.md`** — фазированный план R0–R6 (что делать дальше).
- **`docs/CONTENT_PLAN.md`** — pillar+spoke по кластерам + очередь первых ~30 страниц.
- **`docs/MACEDONIA-ANALYSIS-2026-06-21.md`** — веб-проверенная рыночная база.
- **`docs/memory/`** — проектная память (пока пустая).
- **`docs/_georgia-reference/`** — архив документов Georgia Guidebook (как шаблоны).
