# North Macedonia Guidebook — Roadmap (R0–R6)

**Дата:** 2026-06-21 · **Где мы:** движок перемоделирован под Македонию (англ. слаги, enum'ы, i18n en/ru/uk, бренд, `build`/`check`/`test` зелёные); **коллекции контента пусты**, домен не куплен, на GitHub CI заблокирован биллингом. **Куда:** запуск + контент + монетизация по стратегии EN-first/Tier-1 (`docs/STRATEGY.md`), факты — `docs/MACEDONIA-ANALYSIS-2026-06-21.md`.

> **Критический путь запуска:** домен (O1) + биллинг CI — корневые блокеры → Cloudflare-деплой + кастомный домен + GSC (живой индексируемый сайт) → первый кластер контента → монетизация → масштаб → дистрибуция/реклама → ростовая петля.

> **Сквозной множитель данных:** до подключения GSC/аналитики весь приоритет — **[ОЦЕНКА]**. Первый шаг после запуска (GSC + атрибуция /go/ по SubID) переводит стратегию в **[ДАННЫЕ]** и определяет, что писать дальше.

## Сквозные правила (держать на всех фазах)
- **Парность ru/uk** (`check-parity`) — узел зависимости от **O4**: гейт требует пару ru+uk (en по желанию). Если RU откладывается — зафиксировать, какая пара обязательна на старте, иначе публикация валит гейт.
- **YMYL-осторожность** (визы/налоги/ВНЖ): факты только из первоисточников (`mfa.gov.mk`, `ujp.gov.mk`, `customs.gov.mk`, `stat.gov.mk`, PwC) + видимый дисклеймер. Зашитые ловушки: **РФ нужна виза** (с 21.03.2022); **номад-визы нет**; налог плоский 10%; денар ~61,5/EUR.
- **GSC → [ДАННЫЕ]:** без GSC/атрибуции приоритет — догадки. После запуска приоритизировать кластеры по реальному спросу (MON-R3-2 + DIST-R4-02).
- **Честные оговорки рынка:** ~1,34 млн туристов (в ~4× меньше Грузии) → потолок аффилиата ниже, локальные размещения — слабый канал; резкая сезонность (аффилиат пикует Q3) → приоритет несезонной релокации + evergreen-маршрутов. Основной доход — **медийка Tier-1 + аффилиат**.
- **Перф/CSP-дух:** перф-бюджет (LCP<2.0/CLS<0.1/JS<50КБ) — гейт. Дисплей и GA4 тянут consent и расширение CSP — только по решению O5.

## Ближайшие 5 шагов (старт прямо сейчас)

4. **Я + владелец:** `docs/CONTENT_PLAN.md` (pillar+spoke + очередь первых ~30 страниц + интерлинк-карта).
5. **Я:** demo-тройка (эталон качества) + перенастройка `partners.json`/SEO-обвязки под Македонию; параллельно владелец подключает Cloudflare-деплой и кастомный домен.
