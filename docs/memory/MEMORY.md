# Memory index — North Macedonia Guidebook

Ключевые решения и состояние проекта. Архив исходного движка (референс): `docs/_georgia-reference/memory/`.

## Гео и позиционирование
- **EN-first / Tier-1**: английский — ведущий язык (на корне `/`); ru → `/ru/`, uk → `/uk/` — вторичны. Аудитория первично Запад.
- **Якоря**: Охрид (Ohrid, ЮНЕСКО) + Скопье (Skopje, столица).
- **Валюта**: македонский денар **MKD** (де-факто привязан к евро, ~61,5/EUR).
- **Балканская сеть**: сайт — часть сети (Албания / Черногория / Хорватия); кросс-линки через `NetworkLinks` (дремлет до запуска соседних сайтов).

## Контент-модель
- **Город = статья `category=cities`** (англ. слаг). Коллекция `cities` в конфиге есть, но НЕ используется для наполнения/роутинга.
- Тройки en/ru/uk с одинаковым slug; парность проверяет `check-parity`.
- **8 статистических регионов** в enum `region`: Skopje / Polog / Eastern / Northeastern / Southeastern / Southwestern / Vardar / Pelagonia.
- Партнёрки только через `/go/{partner}` с `rel="sponsored nofollow noopener"`; языковая сегментация `byLang` (EN→глобальный партнёр, RU/UK→дружелюбный к СНГ).

## Механика (не ломать)
- **Новости 10/2 дня**: `NEWS_SECTION_WINDOW=10` (раздел хранит 10 дней), `NEWS_HOME_WINDOW=2` (блок на главной 2 дня), ежедневный `daily-news-rebuild.yml`.
- Погода: Скопье/Охрид/Битола/Маврово (open-meteo). RSS: `/rss.xml`, `/ru/rss.xml`, `/uk/rss.xml`.
