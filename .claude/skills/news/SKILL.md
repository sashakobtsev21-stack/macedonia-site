# /news — дайджест новостей (команда → превью → «ок» → публикация)

Цель: по одной команде принести владельцу **готовый отобранный дайджест** свежих
тревел-новостей Северной Македонии и опубликовать **только то, что он одобрил**. Никакой
слепой автопубликации. Источник правды по продукту — SPEC.md; правила — CLAUDE.md.

**Жёсткие принципы (нарушение = бан/искажения):**
- **Правило 4 — факты не выдумывать.** Каждая новость = пересказ РЕАЛЬНОГО материала с источником в `sources`. Не подтверждено источником → не публикуем (или `<!-- TODO -->`). Никаких выдуманных цифр/дат/цен.
- **🔴 ДАТЫ И ВРЕМЯ — сверять по источнику ТЕКУЩЕГО года (критично!).** Любую дату/время/число бери только из материала, чья СОБСТВЕННАЯ дата публикации = текущий год. **Типичный прокол:** дату ежегодного события (напр. Охридское лето, Galičnik Wedding, Скопский джаз-фестиваль) берут из прошлогодней статьи → публикуют прошлогодние числа. Для **ежегодных событий** (фестивали, сезоны) НИКОГДА не копируй даты из прошлогодней статьи — найди подтверждение дат именно этого года в свежем источнике (новость с датой публикации этого года / официальный портал события). Если дату текущего года подтвердить нечем — НЕ публиковать дату (или новость).
- **§15 копирайт/ToS.** Чужой текст НЕ копировать дословно — только свой пересказ + ссылка на первоисточник. Заголовки тоже свои.
- **Фото обязательно** у каждой новости (cover ≤200КБ webp, легальное: Unsplash/Commons с атрибуцией).
- **Языковая тройка** en+ru+uk, одинаковый slug, парность (en — язык по умолчанию, EN-first).
- **НЕ публиковать без «ок» владельца.** До одобрения всё лежит `draft:true`.

## Аргументы
`/news [N] [окно]` — `N` = сколько новостей в дайджесте (по умолчанию **топ 3–5**),
`окно` = период (по умолчанию **последние 24 ч**). Без аргументов — дефолты.

## Тематика (= фильтр мусора). Берём ТОЛЬКО тревел-релевантное
**Беру** (влияет на поездку туриста):
- въезд / визы / границы / пограничные правила (вкл. отдельную ремарку для граждан РФ — см. ниже);
- авиа: рейсы, новые направления, отмены, аэропорты (**Скопье SKP**, **Охрид OHD**; Wizz Air — основной перевозчик в SKP);
- цены / валюта / **курс денара (MKD)** / налоги и сборы для туриста;
- погода / стихия / ЧС / безопасность / закрытия дорог и перевалов (Маврово, Галичица, перевалы к Охриду);
- открытия и закрытия мест, отелей, транспорта, музеев, нацпарков;
- крупные события, фестивали, праздники с датами (Охрид, Скопье, Битола).

**Выкидываю:** внутренняя политика, выборы, криминал, спорт, светская хроника,
бизнес-сводки без связи с туризмом, реклама, кликбейт, «топ-10»-листиклы, гороскопы.

## Источники (правит только владелец)
Издания (EN): **Balkan Insight**, **Meta.mk (EN-версия)**, **Total Macedonia News**, **Republika English** + балканские EN-СМИ.
**Google News RSS** по запросам: `North Macedonia travel`, `Skopje airport`, `Ohrid airport`,
`North Macedonia visa`, `North Macedonia border`, `North Macedonia tourism`.
Официальные: **МИД Северной Македонии (mfa.gov.mk)** — визы/въезд; **аэропорты Скопье/Охрид (TAV Macedonia)**, **Wizz Air**; **МВД (mvr.gov.mk)** — границы/регистрация в полиции; **Народный банк РСМ (nbrm.mk)** — курс денара.
_Инструменты сбора: `WebSearch` (с указанием свежести) + `WebFetch` (открыть материал и
прочитать факты). Если источник недоступен — пропусти, не выдумывай._

**Визовые реалии (YMYL — только из официального источника, без выдумок):** US/UK/EU/Украина — безвиз 90/180; **гражданам РФ нужна виза** (с 21.03.2022; возможны исключения для держателей действующих виз US/UK/CA/Шенген — проверять по mfa.gov.mk на момент новости). Формальной «визы цифрового кочевника» в Македонии НЕТ — не повторять этот миф. Любую визовую деталь публиковать только со ссылкой на mfa.gov.mk или иной первоисточник.

## Шаги (по порядку, гейты не пропускать)

1. **Сбор.** Пройди по источникам через WebSearch/WebFetch, собери кандидатов за окно
   (по умолч. 24 ч). Для каждого зафиксируй: заголовок, суть, дату, URL первоисточника.

2. **Фильтр мусора — 3 ступени (финальный судья — ты, не регэкспы):**
   - **релевантность** строго по списку «беру» выше — всё остальное отсев;
   - **дедуп/кластеризация** — одну историю из нескольких источников схлопни в один пункт, возьми лучший первоисточник (официальный > крупное издание > агрегатор);
   - **анти-кликбейт/реклама** — выкинь рекламу, пустые и кликбейт-заметки.
   Отбери **топ N** по важности для туриста (свежесть + влияние на поездку).

3. **ПРЕВЬЮ владельцу — СТОП, жди ответа.** Покажи дайджест-таблицу: `№ | тема-тег |
   заголовок (en) | 1–2 строки сути | источник (ссылка) | фото (что поставим)`.
   Кратко перечисли, что отсеял и почему (1 строка). **Ничего пока не пишем в контент.**
   Жди: владелец скажет, какие брать («эти — да», «эту убрать», «добавь…»).

4. **После «ок» — по каждой одобренной новости:**
   - **Скелет:** `npm run new -- news <slug> --title "…"` (slug — латиница-дефис, описательный, стабилен навсегда; при риске коллизии добавь месяц, напр. `-june-2026`). Создаст тройку en/ru/uk `draft:true` + папку фото.
   - **Рерайт en (язык по умолчанию, основной), затем ru/uk**, своими словами, НАСЫЩЕННО: заголовок ≤60, description ≤155; **3–5 коротких разделов** с КОНКРЕТНЫМИ выверенными фактами (числа, даты, время, расписание, цены-если-есть) + «что это значит для поездки» + раздел с более широким контекстом (смежные события/рейсы/тренды по теме, всё со своим источником). Несколько источников в `sources:` (не один). `publishedAt`=наша дата публикации. **≥4 внутренние ссылки** на наши статьи (гейт `check-interlinks` требует ≥2, но цель — насыщеннее): профильный путеводитель/раздел + соседи кластера (город, транспорт, визы, аренда, «когда ехать» и т.п.). По возможности **1–2 инлайн-figure** с уникальным фото (не дублировать кадры). Партнёрские — только через `/go/{partner}?c=<slug>`, по теме и не натужно. НЕ раздувать в лонгрид — это всё ещё новость, но плотная и полезная, а не «на отвяжись».
   - **Фото:** подбери легальное по теме — `node scripts/unsplash-candidates.mjs "<тема>"` или `node scripts/commons-candidates.mjs "<тема>"`, **отбери глазами** (Read рендерит), собери `build-unsplash.mjs`/`build-gallery.mjs` → cover webp ≤200КБ в `public/images/<slug>/` (или общий кадр города: Скопье/Охрид/Битола). `coverCredit` = автор + лицензия. Нет точного кадра события — жанровый кадр города/темы, в `alt`/кредите честно (это иллюстрация, не само событие). **Каждое фото уникально** — не тащить кадр из другой статьи.
   - **Перевод:** `uk-translator` → uk, и сверка ru (Agent). Паритет: тот же slug, те же числовые факты, те же `gallery`/`cover` src.

5. **Гейт.** `npm run qa` → дождись **GO** (чинит парность/H1≤60/desc≤155/≥2 ссылки/cover≤200КБ). NO-GO — не публикуем, чиним.

6. **Публикация.** Сними `draft:false` у одобренных (остальные кандидаты НЕ коммить — либо удали скелеты, либо оставь `draft:true`). Один коммит на батч → пуш в `main`. Cloudflare соберёт; свежесть/ретеншн уже работают (**окно раздела 10 дней `NEWS_SECTION_WINDOW`, блок на главной 2 дня `NEWS_HOME_WINDOW`, ежедневный `daily-news-rebuild.yml`**).

7. **Синхронизация доков** (правило дисциплины CLAUDE): отметь прогон в `PROGRESS.md`/`HANDOFF.md`, затем `git commit` + `git push` в `main`.

## Запреты
Не публиковать без «ок» владельца. Не копировать чужой текст/заголовки дословно (§15).
Не выдумывать факты (правило 4). Не публиковать новость без фото и без источника.
Не публиковать половину языковой тройки. Не тащить политику/криминал/спорт/рекламу.
Не повторять опровергнутые мифы (безвиз для РФ; «номад-виза Македонии»). Прямые партнёрские URL запрещены — только `/go/`.
